# Triple Espresso

Página web para la cafetería de especialidad de Triple Peaks, esta web fue desarrollada con HTML y CSS, en base a diseño del brief.

# Caracteristicas del proyecto

-HTML semántico
-Flexbox
-Posicionamiento
-Inframe
-Transicion
-Formulario

# Funcionalidad

-La web posee un encabezado con nav links direccionales a cada bloque de sección de la página.

-El encabezado da una bienvenida a la web y muestra los horarios de atención así como la dirección del establecimiento.

-En el bloque Recetas, se presentan videos de youtube que el usuario puede dirigirse, ver o maximizar para ver una receta cafetera recomendada.

-En el bloque Reserva una mesa se presenta un formulario con el proposito de hacer una reservación en la cafetería, donde se solicita datos de contacto de la persona, el día y hora que quiere reservar y el número de comensales que llegarán.

-El pie de página, el contacto de las redes sociales de la cafetería.

Aunque el sitio es estático, está estructurado semánticamente con la metodología BEM por lo que será fácil incluir futuras integraciones dinámicas o interactivas.
